{# BUTTON COMPONENT MACRO ====================== A reusable button component with multiple variants and sizes. USAGE: ------ 1. Import the macro in your template: {% from 'components/button.html' import button %} 2. Use the button macro: {{ button('Click Me', href='/some-page', variant='primary', size='large') }} PARAMETERS: ----------- @param label (required) - The text displayed on the button @param href (optional, default="#") - The URL the button links to @param variant (optional, default="primary") - The button style variant @param size (optional, default="medium") - The button size AVAILABLE VARIANTS: ------------------- - "primary": Blue button (#0021A5) - Use for main call-to-action buttons - "secondary": Gray button (#6C757D) - Use for secondary actions - "outline": Transparent with blue border - Use for tertiary actions AVAILABLE SIZES: ---------------- - "small": 8px/16px padding, 14px font - Compact buttons - "medium": 12px/24px padding, 16px font - Standard buttons (default) - "large": 16px/32px padding, 18px font, bold - Prominent buttons EXAMPLES: --------- {{ button('Submit Form', href='/submit', variant='primary', size='large') }} {{ button('Cancel', href='/back', variant='secondary') }} {{ button('Learn More', href='/docs', variant='outline', size='small') }} {{ button('Default Button') }} #} {% macro button(label, href="#", variant="primary", size="medium") %} {{ label }} {% endmacro %}
